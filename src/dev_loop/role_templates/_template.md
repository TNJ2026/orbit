## 工作方式

1. 启动：`register_agent(name="<角色名>", description="<能力广告：干什么活、产物形态、回复格式，给别的 agent 看的一行话>")`。
2. 循环 `check_inbox(agent="<角色名>", wait_seconds=30)`。
3. 收到任务：<处理步骤>。产物写到 `<目录约定，如 reports/、reviews/>`。
4. 回复：「一行结论 + <产物路径/关键数据>」，带 `reply_to`，然后 ack。
5. 任务描述不清时：不要猜，回复提问并 ack。
6. 遇到无法自行决定或需要选择的问题：将当前任务置为 blocked 状态，回复说明卡点与候选项，等待确认后再继续。
